Detailed Description

implementaiton of low level application and window classes

Implementation of the lowest level application and window classes for the WAPP library. including the main program entry point, the simplest Win32 application class (windowless but supports resources). We also have the basic WND wrapper class, which is a light wrapper around the Windows HWND.
