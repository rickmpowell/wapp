Detailed Description

The base application class.

The base application handles COM initialization, Direct2D initialization, and provides basic access to the application via the instance handle. The instance handle is how resources are accessed.

The app represents the instance of the Windows application, without a window attached to it. It basically corresponds to a Windows instance, which means resources are loaded through this object.

The APP initializes the COM subsystem for the application.
