Detailed Description

Colors.

This is just a light-weight wrapper around the DirectX D2D1_COLOR_F type. We include several standard color definitions, and common operations on colors that should make it easier to modify colors to get interesting variants based on an original color.

float CompBlend ( float  a,
float  b,
float  alpha 
)
inline

Blends two color components, with gamma-correction.

Note that this is just an approximation. We round the standard gamma correction from 2.2 to 2, which simplifies the math. The actual gamma corrected blend should be: 

pow((1-alpha) * pow(a, gamma) + alpha * pow(b, gamma), 1/gamma)
