Detailed Description

Layout engine.

This is a base class for layout engines. The engine is typically attached to a container window, and the Measure and Locate methods are called during the container's Layout method.

The items/controls in the container provide some standard methods and information for the layout engine to use. In particular, items should have a "role", and a series of "behaviors". The layout engine in the container will use these to determine how to size and position each item.

To aid in control layou, each controls should also provide an intrisic size, which would provide the "natural" size for the item.

Note that roles and behaviors are typically assigned by the container, although in some cases it may make sense for a control type to have a default.
