Detailed Description

template<typename T>
class observer_ptr< T >

A non-owning smart pointer.

A simple non-owning smart pointer that just wraps a raw pointer. This is mostly for documentation purposes, to make it clear that the pointer is not owned by the class or function that receives it. It has no special behavior beyond that.

TODO: this could be replaced with std::observer_ptr when we move to C++23
