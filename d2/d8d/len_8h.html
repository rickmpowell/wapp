Detailed Description

Layout engine.

A rudimentary and experimental layout engine for aiding in the automatic layout of dialog boxes.

The engine is derived from the CSS flexbox model and uses the same terminology and concepts. It is not complete, however.

◆ LEROLE

enum class LEROLE : uint32_t
strong

The role of an item in the layout engine.

The role is typically assigned by the container to each item. It indicates the purpose of the item within the container, and is used by the layout engine to determine how to size and position the item.

Roles are typically specific to a particular container type. A toolbar will have different roles than a dialog box, for example.
