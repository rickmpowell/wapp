Detailed Description

template<typename D, typename WAPP>
class CMD< D, WAPP >

Command base class.

Keeps some standard information around that is useful for all the commands in the applciation.

Typical usage: 

class CMDFOO : public CMD<CMDFOO, WAPP> 
{
public:
    CMDFOO(WAPP& wapp) : CMD(wapp) { }

    virtual int Execute(void) override 
    {
        wapp.DoSomething();
        return 1;
    }
};

void WAPP::RegisterMenuCmds(void)
{
    RegisterMenuCmd(cmdFoo, new CMDFOO(*this));
}
