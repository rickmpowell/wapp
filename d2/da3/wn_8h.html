Detailed Description

Windows.

A rectangular area on the screen, arrranged in a parent- child tree structure. The top-level root represents the client area of the Windows HWND.

The implementation takes advantage of the way Direct2D updates on a Windows screen, with all drawing done off-screen and then swapped to the screen at the end. This allows for flicker-free drawing in sub-optimal updates.

These objects do not have a user interface associated with them, it is purely visible. To process mouse of keyboard input, multi-inherit this with the various input type classes.
