Detailed Description

A utility class that reads text files as a sequence of lines. Handles UTF-16, UTF-8, and regular ASCII files. Permits a push operation that returns strings back into the stream which allows for code that needs a line look-ahead.

The strings returned as lines are UTF-8. Line end marks are stripped. Will return empty lines
